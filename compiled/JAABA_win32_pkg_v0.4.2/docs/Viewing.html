Movie Viewing Options

Zoom-in, Zoom-out, and Pan buttons are above the movie previewer. While playing the movie, to keep the animal at the center use View -> Zoom -> Center axes on current target. View -> Zoom -> Keep target in view does not move the axes unless the animal moves outside the preview window, in which case it moves the axes to keep the animal in the view. View -> Zoom -> Static View does not move the axes even if the animal goes outside the preview window. You can set the movie playback speed using View -> Preview Options..
