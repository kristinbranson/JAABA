Creating a new project



At the initial user interface, you can add a new project or continue with a previously added project. To add a project click on the + at the bottom of the project list. In the pop-up window, specify the projects name and a path to an xml file to save the project configuration. You can specify an existing xml file, in which case it'll use the settings from that file.

In the demo version, "FlyBowl_Chase" project should exist in the project list and should also be selected. This project is used to train a chase detector. You can add or remove projects using the + and - signs at the bottom of the project list. After selecting the project, you add movies/experiments to view and annotate. Using the Add.. button at the bottom browse to the experiment directory and add it. After adding one or more of the experiment directories click Done.
