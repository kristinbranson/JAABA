Adding Experiments to the Current Project


After selecting the project, you can add movies/experiments to view and label. Using the Add.. button at the bottom, select the experiment's directory and click open. After adding one or more of the experiment directories click Done. You have to add the experiments each time you start JAABA. To add multiple experiments you can create a text file with the location of the experiments (one experiment per line) and use Add List.. button to add all the experiments.

You can add/remove experiments later within a JAABA session using File -> Edit files... .
