Adding Experiments to a JAABA Project


Any time after you have created a JAABA project, you can add videos (termed experiments) to the project to view and label. To do this, select the File->Change Experiment List... menu. This will bring up the Change Experiment List dialog:

The list on the left shows the experiment directories that are currently part of the project. The list on the right shows information about the files in the currently selected project: which files exist and when they were last modified. You can modify the experiment directories included in the project as follows:

  • You can add experiments by pushing the Add... button and selecting the directory or directories you want to add:
  • You can remove the selected experiment from the project by pushing the Remove button.
  • If you have a text file containing a list of experiments you want to include in the project (one experiment per line), you can add all these experiments by selecting Add List...
