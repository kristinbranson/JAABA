Installation and Starting JAABA

REQUIREMENTS

Linux (64-bit), Windows (32- or 64-bit), or MAC (64-bit) systems with at least 4GB RAM. JAABA uses parallel computation, and will be faster on multicore machines. If you are running JAABA in Matlab using the source code, then you will require Parallel Computation Toolbox. JAABA is a computationally intensive program, and would work better on more powerful machine.

INSTALLATION

Download the JAABA packwage for you system from http://sourceforge.net/projects/jaaba/.

The executables are compiled Matlab programs and do not require Matlab, but require the Matlab Compiler Runtime (MCR)

Install the Matlab Compiler Runtime (MCR) for R2012a. MCR installers and the instructions are available at Mathworks website.

You don't need MCR if you'll run JAABA in Matlab from the source code.

STARTING UP

On Windows: From JAABA/Win32 or JAABA/Win64 folders depending on your system, run the executable JAABA_win.exe
On 64-bit MAC: From JAABA/MAC64, run run_JAABA_mac.command file. Do not run the JAABA_mac appication.
On 64-bit Linux: From JAABA/Linux64, run the run_JAABA_linux.sh file from the command line in a terminal. You need to give the path to installed MCR as the argument.
From source code in Matlab: Change directory to JAABAROOT/perframe and run StartJAABA.m

Loading the MCR is slow, and hence once you run the executable it may take some time for JAABA to appear.
