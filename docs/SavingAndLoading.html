Saving and Loading

Saving Labels and Classifiers

The labels can be saved using File -> Save Labels. The labels for all the experiments get saved in the experiment directory with the name that specified in the configuration file (value of the file.labelfilename parameter). The saved labels are automatically loaded when the experiment is used with the same project in a new JAABA session. If you add the same experiment, but select a different project, labels for the current project won't be loaded.

Similarly, File -> Save classifier and labels will save the current classifier and the labels. The labels are saved to their default location. For the classifier, JAABA will ask for a location to save it. A saved classifier can be loaded using the Load and Load w/o exps buttons when starting a new JAABA session. Load will load the classifier alongwith with movies/experiments that were used to train the classifier while Load w/o exps loads the classifier without any of the experiments.

Predicting on Whole Movies

After training a classifier, the whole trajectory of a animal, the whole experiment, or all the experiments can be classified by Classifier -> Classify. Current animal computes the predictions for the whole trajectory of the current animal. Current Experiment -> Don't Save computes the predictions for all the animals in the current experiment but does not save the prediction results (scores). Current Experiment -> Save to default saves the scores in the experiment directory to the file specified by the file.scorefilename parameter in the project configuration. Current Experiment -> Save to different asks the user for a location to save the scores. You can also predict on all the experiments without saving using All Experiments-> Don't Save. All Experiments -> Save to default saves the scores to the scores file specified by file.scorefilename in the projects configuration. All Experiments -> Save to different will ask for a score file name to save the scores. The scores are saved in the experiment directory.

The predictions computed using Current Experiment and All Experiments are shown at the bottom of the prediction timeline (the Loaded option in the drop down menu).

Previously saved scores can be loaded using File -> Load Scores -> For current experiment from default location or For all experiments from default location. Other options in File -> Load Scores can be used to load saved scores from non-default locations.
