JAABA Project Overview

All data associated with a given behavior classifier is stored in a JAABA project file with the extension .jab.

For each behavior, you need a separate JAABA project. This includes the:

  • Project parameters, such as the behavior name and what type of animal is being labeled.
  • Trained behavior classifier, which inputs information from the animals' trajectories and predicts the behavior class.
  • Training data labels and locations.

Behavior Classifier

The most recent behavior classifier trained using JAABA is also saved in the JAABA project file. The classifier is what is used to automatically predict the behavior of a given animal in a given frame. This classifier can be used to classify the behavior of the animals in a new video outside of your training data set.

Training Data Labels and Locations

The data that has been used for training the classifier is also stored in the JAABA project file. This includes:
  • The locations of the experiment directories added to this project (see Adding Experiments to the Current Project).
  • The behavior labels manually entered by the user for any experiments, animals, and frames in this project.

Creating a New Project

You can create a new project either by
  • Selecting Start a new project in the initial dialog when JAABA starts
  • Selecting the File->New menu at any time.
This will cause the New... project dialog box to open:
New Project Dialog
In this dialog, you can specify the Project Parameters described above. After you have set the JAABA project parameters, you can add experiments to the project by selecting the File->Modify experiment list menu. You can create a new project when JAABA starts up throught the Project Configuration Dialog.

Project Configuration Dialog

For each behavior, you need a separate JAABA project. To add a new project, click on the New Project button in the initial JAABA edit project and experiments interface.

Screen capture of Edit Projects and Files interface

This will open a Project Setup window. In this window, select the animal type for the project using the Target Type drop down menu and specify the name of the behavior for the project and the files that will be used by JAABA for this project.

Click Done to save the project and close the window. JAABA will ask for a location to save the project as a mat file. Once you create a project, you can continue working on this project, use Select Project button in the Edit Project and Files window to select this project file. Also, you can edit an existing project using Edit Project button after you select the project. When creating a new project, you can copy the settings from an existing project using the Copy Settings From button.
