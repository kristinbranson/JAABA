JAABA Project Overview

All data associated with a given behavior classifier is stored in a JAABA project file with the extension .jab. This includes the:
  • Project parameters, such as the behavior name and what type of animal is being labeled.
  • Trained behavior classifier, which inputs information from the animals' trajectories and predicts the behavior class.
  • Training data labels and locations.

The following parameters must be set for each JAABA project. Information on the experiment directory structure referred to below can be found here.

Behavior Classifier

The most recent behavior classifier trained using JAABA is also saved in the JAABA project file. The classifier is what is used to automatically predict the behavior of a given animal in a given frame from the When JAABA starts, it will ask you whether you want to:
  • Start a new project
  • Open an existing project in training mode
  • Open an existing project in ground-truthing mode
Training mode is the standard mode

If you choose to start a new project the New... dialog box will open. You can create a new project when JAABA starts up throught the Project Configuration Dialog.

Project Configuration Dialog

For each behavior, you need a separate JAABA project. To add a new project, click on the New Project button in the initial JAABA edit project and experiments interface.

Screen capture of Edit Projects and Files interface

This will open a Project Setup window. In this window, select the animal type for the project using the Target Type drop down menu and specify the name of the behavior for the project and the files that will be used by JAABA for this project.

Click Done to save the project and close the window. JAABA will ask for a location to save the project as a mat file. Once you create a project, you can continue working on this project, use Select Project button in the Edit Project and Files window to select this project file. Also, you can edit an existing project using Edit Project button after you select the project. When creating a new project, you can copy the settings from an existing project using the Copy Settings From button.

Using Output of Previously Trained Classifiers

In JAABA, you can use output of previously trained behavior detectors as inputs for the current behavior. Use the Add and Remove buttons to specify the list of saved classifiers whose output you want to use as inputs. Note that the scores/predictions for the classifiers should already be generated for all the experiments that you want to use for this project.
